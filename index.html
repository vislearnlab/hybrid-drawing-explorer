🧬 Hybrid Drawing Explorer

Children's drawings of hybrid concepts ("a bike bee", "a tiger frog"…) and their pure constituents, in CLIP space.
ⓘ What do the metrics mean?

Each drawing becomes a 512-number CLIP "fingerprint". Each pure concept's average fingerprint is its centroid. A hybrid of A + B is placed relative to A's and B's centroids:

Interpolation weight (blend) — where the drawing falls on the line from constituent 1 (0) to constituent 2 (1). 0 = looks like the first, 1 = the second, 0.5 = even blend.

Residual (emergent features) — how far off that line it sits. Low = a clean mash-up of the two; high = features that are neither parent (something new the child invented).

Favored constituent — same as blend, colored so you can see which parent each leans toward.

Similarity to constituents — how much it looks like each parent on its own (0–1).

UMAP / t-SNE — two ways to flatten the fingerprints into a 2-D map; nearby dots look alike. The toggle switches between them.

Tip: set hybrids to None to see only the pure categories.
